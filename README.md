# eslint-config-jnmorse

Personal Eslint Configuration.

## Packages Used

- eslint
- eslint-plugin-react
- eslint-plugin-import
- eslint-plugin-react-a11y

## Install Instructions

- NPM
```sh
npm i -D eslint eslint-config-jnmorse
```

- Yarn
```sh
yarn add -D eslint eslint-config-jnmorse
```

## Configuration

*.eslintrc.json*
```json
{
  "extends": ["jnmorse"],
  "rules": {}
}
```
