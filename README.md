# eslint-config-jnmorse

Personal Eslint Configuration.

## Packages Used

- eslint
- eslint-plugin-react
- eslint-plugin-import

## Install Instructions

- NPM
```sh
npm i -D eslint jnmorse/eslint-config-jnmorse
```

- Yarn
```sh
yarn add -D eslint jnmorse/eslint-config-jnmorse
```

## Configuration

*.eslintrc.json*
```json
{
  "extends": ["jnmorse"],
  "rules": {}
}
```
